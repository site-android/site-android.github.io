Android e suas versões

São muitos, e curiosos, os nomes dados para cada versão do Android, sistema operacional para celulares criado pelo Google. Normalmente, os nomes são de doces ou guloseimas americanas. Mas qual é exatamente a diferença entre os celulares com Android? Além de mudanças na performance, cada versão trouxe consigo algumas novidades bem interessantes, e você vai conhecer um pouco de cada a seguir.
